# VenkataTaskHubApp

Native iPhone app project for the same mortgage and compound-interest tools from the website.

## Generate the Xcode project

From this folder:

```bash
xcodegen generate
```

This creates `VenkataTaskHubApp.xcodeproj`.

## Open in Xcode

```bash
open VenkataTaskHubApp.xcodeproj
```

## Notes

- The app is written in SwiftUI.
- It uses Apple's `Charts` framework for the compound-interest visualization.
- You need full Xcode installed, not just Command Line Tools, to build and run the iOS app.
